My API Project #3: URL shortener

How it works:
    1) You can pass a URL and it will return a JSON with a shortened version of the URL
    2) If you click on either the original version of the URL or the short version, it will redirect you to the original URL

Example usage:

https://little-url.herokuapp.com/new/https://www.freecodecamp.com

Example output:

{
   "long-url": "https://www.freecodecamp.com",
   "short-url": "https://url-shortener-est.herokuapp.com/Skc7Qbblb"
}
